# kubectl

Kubernetes cluster operations via the `kubectl` CLI. List pods, deployments,
events, namespaces, view logs, and describe resources.

## Tools

| Tool | Description |
|------|-------------|
| `k8s/get-pods` | List pods in a namespace |
| `k8s/get-deployments` | List deployments |
| `k8s/get-events` | List cluster events |
| `k8s/logs` | View pod logs |
| `k8s/get-namespaces` | List namespaces |
| `k8s/describe` | Describe any resource |

Requires `kubectl` configured with cluster access on the host.
